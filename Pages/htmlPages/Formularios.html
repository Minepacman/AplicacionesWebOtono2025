Formularios en HTML

Un formulario es un elemento que permite recoger datos introducidos por el usuario. Se utilizan para conocer opiniones, dudas, introducir pedidos a través de la red, etc.

Un formulario está formado por etiquetas, campos de texto, menús desplegables y botones.

La Etiqueta <form>

Los formularios se insertan a través de las etiquetas <form> y </form>. Entre dichas etiquetas habrá que insertar los diferentes objetos.

Atributos Principales

  • action: Indica la dirección de correo o del programa que procesará el formulario.
  • enctype: Indica el modo de cifrado. Por defecto es application/x-www-form-urlencoded.
  • method: Indica cómo se transfieren las variables (GET o POST). 
<form action="mailto:formularios@servidor.com" method="post" enctype="text/plain">
    ...
</form>

Métodos: GET vs POST

Método GET

Se utiliza cuando no se van a producir cambios en ningún documento o programa (solo lectura).

Ejemplo: Consultas a bases de datos o buscadores.

Método POST

Se utiliza cuando sí se van a producir cambios, como almacenar datos.

Ejemplo: Registrar un usuario, subir un archivo, guardar un pedido.

Áreas de Texto <textarea>

Permiten insertar varias líneas de texto, ideales para comentarios.

  • rows: Número de líneas visibles (alto).
  • cols: Número de caracteres por línea (ancho).
  • name: Nombre para identificar el campo.
<textarea name="comentarios" cols="30" rows="3">
Escribe aquí...
</textarea>

Campo de Texto e Inputs <input>

La etiqueta <input> define la mayoría de los campos de entrada.

Atributos comunes

  • type: Tipo de entrada (text, password, etc).
  • name: Nombre del campo.
  • size: Ancho visual en caracteres.
  • maxlength: Número máximo de caracteres permitidos.

Tipos de Input

1. Texto y Contraseña 
<input type="text" value="Campo Texto">
<input type="password" value="secreto">
2. Checkbox y Radio 
<!-- Checkbox (Selección múltiple) -->
<input type="checkbox" checked> Acepto

<!-- Radio (Selección única) -->
<input type="radio" name="grupo1" value="opcion1"> Opción 1
<input type="radio" name="grupo1" value="opcion2"> Opción 2
3. Otros (File, Hidden) 
<input type="file">
<input type="hidden" value="dato_oculto">

Botones

Se definen mediante <button> o <input type="...">. El atributo value indica el texto que mostrará.

  • type="submit": Envía el formulario.
  • type="reset": Restablece el formulario a su estado inicial.
  • type="button": No realiza acción por defecto (para JavaScript).
