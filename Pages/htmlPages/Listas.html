Listas en HTML

Las listas son esenciales para organizar información de manera estructurada. HTML nos ofrece principalmente dos tipos: listas donde el orden no importa (con viñetas) y listas donde el orden es secuencial (con números).

Listas No Ordenadas <ul>

Las listas no ordenadas van dentro de la etiqueta <ul> y su cierre </ul>. Cada punto que queramos añadir a la lista lo haremos dentro de la etiqueta <li> (List Item).

Atributo Type

Si no le indicamos nada a la etiqueta, esta generará un punto (bullet) de forma automática. Pero si queremos definir nosotros mismos el símbolo, podemos hacerlo gracias al atributo type en la etiqueta <li> (aunque actualmente se prefiere hacerlo con CSS, este atributo es parte de la especificación clásica).

  • type="disc": Un punto lleno (por defecto).
  • type="circle": Un punto hueco (fondo blanco).
  • type="square": Un cuadrado.
Ejemplo de Código: 
<ul>
    <li type="circle">Este es un punto hueco</li>
    <li type="disc">Este es un punto lleno</li>
    <li type="square">Este es un cuadrado</li>
</ul>

Resultado Visual:

  • Este es un punto hueco
  • Este es un punto lleno
  • Este es un cuadrado

Listas Ordenadas <ol>

Las listas ordenadas van enmarcadas dentro de las etiquetas <ol> y </ol>. Cada punto de la lista se escribe con la misma etiqueta que en las no numeradas: <li>.

Al ser listas ordenadas, los símbolos serán números y estos se irán generando automáticamente por orden conforme escribamos nuevos puntos.

Atributo Value

Podemos hacer que un punto específico comience con el número que nosotros queramos gracias al atributo value. Los siguientes puntos se generarán automáticamente partiendo de ese número.

Ejemplo: Iniciar la cuenta en 20 
<ol>
    <li value="20">Este número será 20</li>
    <li>Este número será 21</li>
    <li>Este número será 22</li>
</ol>

Resultado Visual:

  1. Este número será 20
  2. Este número será 21
  3. Este número será 22
