¿Qué es HTML?

Es más una codificación que un lenguaje de programación. Su estructura básica es tan simple que cualquier persona sin principios en programación puede aprenderlo con gran facilidad.

Lo sorprendente de este lenguaje es que proporciona al usuario la información de manera interactiva, haciendo uso del Hypertexto (texto con enlaces hacia otros lugares de la Web o inserciones multimedia). Además, es universal y no depende del sistema operativo que se esté utilizando.

Los documentos HTML se escriben en modo ASCII, haciendo uso de cualquier procesador de palabras.

Estructura y TAGs

El HTML utiliza una codificación genética que hace uso de TAGs o etiquetas. Mediante estas etiquetas es posible separar el contenido del documento de su formato.

Los TAGs son comandos que se especifican en el cuerpo del programa con el fin de darle las características deseadas a la información.

Body: Cuerpo del documento

El cuerpo del documento contiene la información propia del documento, es decir, lo que queremos que se visualice: el texto de la página, las imágenes, los formularios, etc. Todos estos elementos tienen que encontrarse entre las etiquetas de <body>.

Metadatos <meta>

La etiqueta <meta> sirve para definir datos sobre el documento que no son visualizados por el navegador, pero son útiles para buscadores. Se incluyen en el encabezado.

  • charset: Establece el conjunto de caracteres (ej. UTF-8).
  • keywords: Palabras clave para buscadores.
  • description: Descripción del documento.
  • author: Nombre del autor.
  • generator: Programa con el que se creó el documento.

Recursos Externos <link>

Permite establecer una relación con un documento externo. 

<link rel="stylesheet" href="estilos.css">

Atributos importantes:

  • href: La URL del documento a enlazar.
  • rel: Relación con el documento (ej. stylesheet, icon, author, license).
  • type: Tipo MIME del documento (ej. text/css).
  • media: Dispositivo de destino (screen, print, all).

Colores (Hexadecimal y Nombres)

En la etiqueta body es posible establecer el color de fondo a través del atributo bgcolor.

Representación Hexadecimal

Los colores en HTML se representan mediante un número hexadecimal que sigue el formato RGB. Cada color estará representado por un grupo de 6 dígitos precedidos por una almohadilla, por ejemplo: #FFFFFF (Blanco).

Representación por Nombre

Existe otra forma de representar colores sin usar números, utilizando sus nombres predefinidos (ej. blue, red, yellow).

Ejemplo 1: Usando Hexadecimal para fondo azul

<html>
    <body bgcolor="#0000FF">
        <p>hola</p>
    </body>
</html>

Ejemplo 2: Usando Nombre para fondo azul (Equivalente)

<html>
    <body bgcolor="blue">
        <p>hola</p>
    </body>
</html>

Elementos Estructurales HTML5

Etiquetas para organizar la estructura de la página y agrupar contenido.

Contenedores Genéricos

  • <div>: Contenedor a nivel de bloque. Crea un salto de línea y sirve para agrupar secciones grandes.
  • <span>: Contenedor a nivel de línea. No crea saltos, ideal para aplicar estilos a trozos de texto.
  • <iframe>: Marco en línea para incrustar otros documentos (mapas, videos) dentro de la página actual.

Etiquetas Semánticas

Dotan de sentido a la estructura del documento:

  • <header>: Encabezado de un documento o sección.
  • <nav>: Conjunto de vínculos de navegación.
  • <section>: Define secciones como capítulos.
  • <article>: Contenido independiente o autónomo (posts, noticias).
  • <aside>: Contenido al margen relacionado con el principal.
