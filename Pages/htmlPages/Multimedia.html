Imágenes en HTML

Todas las páginas web acostumbran a tener un cierto número de imágenes, que permiten mejorar su apariencia o dotarla de una mayor información visual.

Para insertar una imagen es necesario utilizar la etiqueta <img>. Dicha etiqueta no necesita etiqueta de cierre.

Atributo src

El nombre y ubicación de la imagen ha de especificarse a través del atributo src (source/origen).

Sintaxis: 
<figure>
    <img src="imagenes/robot.gif">
</figure>

Texto Alternativo (alt)

Cuando una imagen no puede ser visualizada en el navegador (cosa que puede ocurrir al especificar mal el valor del atributo src o si el archivo fue borrado), normalmente aparece un recuadro blanco con una X roja o un icono de imagen rota.

Podemos hacer que en lugar de mostrarse el error, aparezca un texto descriptivo gracias al atributo alt.

Ejemplo con error intencional:

Imagina que deseamos mostrar una imagen llamada gatito.gif con el texto alternativo "Imagen gato", pero la imagen no existe en la carpeta.

<img src="imagenes/gatito.gif" alt="Imagen gato">

Imágenes como Enlaces

Como ya conocemos los hipervínculos y cómo insertar imágenes, podemos implementar un hipervínculo que, en vez de mostrar un texto subrayado, muestre una imagen clicable.

La solución es simple y consiste en disponer la etiqueta <img> encerrada entre la marca de comienzo y fin del enlace <a>.

Ejemplo:

Generamos un ejercicio que muestre una imagen que, al ser presionada, realice un salto a otra página.

<a href="pagina2.html">
    <img src="imagenes/foto1.jpg" alt="Pintura Geométrica">
</a>

Video HTML5 <video>

Permite reproducir archivos de video (MP4, WebM, Ogg). Se usa la etiqueta <video> junto con <source> para definir el archivo.

Atributos Principales:

  • controls: Muestra botones de play, pausa, volumen.
  • autoplay: Reproduce el video automáticamente al cargar.
  • loop: Reproduce en bucle infinito.
  • muted: Silencia el video.
  • poster: Imagen previa antes de reproducir.
  • width / height: Dimensiones del reproductor.
<video width="320" height="240" controls>
    <source src="pelicula.mp4" type="video/mp4">
    <source src="pelicula.ogg" type="video/ogg">
    Tu navegador no soporta video.
</video>

En este ejemplo, el video se mostrará con controles, y si el navegador no soporta la etiqueta <video>, se mostrará el texto alternativo.
