Enlaces y la etiqueta <a>

Un enlace, hipervínculo o vínculo, es un elemento que al ser pulsado lleva a una página o archivo. Aquellos elementos (Texto, imágenes, etc.) sobre los que se desee insertar un enlace han de encontrarse entre las etiquetas <a> y </a>.

A través del atributo href se especifica la página a la que está asociado el enlace, es decir, la página que se visualizará cuando el usuario haga click.

Ejemplo: 
<a href="https.cs.buap.mx"> Visita la FCC </a>

Resultado visual:

Visita la FCC

Referencias Absolutas vs Relativas

Existen diferentes formas de expresar una referencia a través del atributo href:

1. Referencia Absoluta

Conduce a una ubicación externa al sitio. La ubicación es en internet; en este caso hay que empezar la referencia por http://, el nombre del dominio y algunas veces el nombre de la página.

2. Referencia Relativa a Sitios

Conduce a un documento situado dentro del mismo sitio, partiendo de la carpeta raíz. Se identifica porque aparece el símbolo / delante del nombre. 

<a href="/t_4_1.html"> Enlace a Tema </a>

Si el documento estuviera dentro de una carpeta llamada tema4 dentro de la raíz: 

<a href="/tema4/t_4_1.html"> Enlace a Tema </a>

3. Referencia Relativa al Documento

Conduce a un documento situado dentro del mismo sitio, pero partiendo del directorio en el que se encuentra el documento actual. No aparece el símbolo / al inicio.

Ejemplo de estructura de carpetas:

root:www
    - Otra
      - obj.html
    - P1
        - P2
            - P3
                 - Act.html

Si el archivo destino está en la misma carpeta:

<a href="documento.html"> Enlace directo </a>

Si está en una subcarpeta llamada tema4:

<a href="tema4/t_4_1.html"> Enlace a subcarpeta </a>

Formato de Enlaces

Los colores de los vínculos pueden especificarse en la etiqueta <body> mediante atributos:

  • link: Color de enlaces sin visitar.
  • alink: Color del enlace activo (al pulsar).
  • vlink: Color de enlaces visitados.
<body link="#F00" vlink="#F09" alink="#F90">

Puntos de fijación (Anclas)

Cuando se tienen documentos extensos, es preferible ir directamente al apartado deseado. Para ello se utilizan las anclas.

Paso 1: Crear el ancla

Se utiliza el atributo name. No es necesario insertar nada entre las etiquetas. 

<a name='miancla'>Texto ancla</a>

Paso 2: Enlazar al ancla

El vínculo debe ser nombre_del_documento#nombre_de_punto. Si el ancla está en la misma página, se puede omitir el nombre del documento. 

<!-- Enlace en la misma página -->
<a href="#miancla">Enlace al ancla</a>

<!-- Enlace desde otra página -->
<a href="t_4_3.html#miancla">Enlace externo al ancla</a>

Destino del enlace (Target)

El atributo target determina en qué ventana va a ser abierta la página vinculada.

  • _blank: Abre el documento en una ventana nueva.
  • _self: Opción predeterminada. Abre en la misma ventana.
  • _parent: Abre en el marco padre.
  • _top: Abre en la ventana completa del navegador.

Es interesante usar _blank cuando la página de destino está fuera de nuestro sitio.

<a href="https://www.buap.mx/" target="_blank">Visita la BUAP</a>

Otros tipos de enlaces

Correo Electrónico

Abre la aplicación de correo electrónico. La referencia debe ser mailto:. 

<a href="mailto:webmaster.ri@correo.buap.mx">Enviar e-mail</a>

Podemos pre-rellenar el asunto (subject):

<a href="mailto:a@a.com?subject=El Asunto">Correo con asunto</a>

Archivos para descarga

Si el atributo href apunta a un fichero comprimido, documento Word, Excel, etc., el navegador pedirá permiso para descargarlo. 

<a href="carta.doc" target="_blank">Descargar fichero</a>

Vínculo vacío

Útil para comportamientos JavaScript. No abre ninguna página. 

<a href="#"> Vínculo Vacío </a>
