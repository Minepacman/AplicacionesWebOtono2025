Temario Completo de CSS

  • Fundamentos

    • Definición y Modelo de Caja
    • Estructura: Block vs Inline-block
    • Inserción:
      • En línea (style)
      • Embebido (head)
      • Externo (link)

  • Selectores y Sintaxis

    • Básicos: Universal (*), ID (#), Clase (.)
    • Relacionales: Hijo (>), Adyacente (+), General (~)
    • Avanzados:
      • Negación (:not)
      • Atributos ([name^="val"])
      • Estructurales (nth-child, first-child)

  • Tipografía y Color

    • Fuentes: font-family, size, weight, style
    • Propiedad Shorthand: font
    • Colores de Texto: Hex, RGB, Nombres
    • Colores de Borde

  • Estilizado Visual

    • Backgrounds:
      • Color, Imagen, Position, Repeat
      • Propiedad Shorthand (Taquigrafía)
    • Bordes: border-radius (Redondeados)
    • Fondos Múltiples

  • Unidades de Medida

    • Absolutas: px, cm, pt
    • Relativas:
      • Al contenedor (%)
      • A tipografía (em, rem)
      • Al Viewport (vw, vh)
