Unidades de Medida en CSS

Las medidas en CSS se emplean para definir la altura, anchura, márgenes de los elementos y para establecer el tamaño de letra. Todas las medidas se indican como un valor numérico seguido de una unidad.

Unidades Absolutas

Las unidades absolutas establecen de forma fija el valor de una medida. Su valor real es directamente el valor indicado y no cambia según el contexto.

  • px: Píxeles (la más común en pantallas).
  • pt: Puntos (usado comúnmente en impresión).
  • cm: Centímetros.
  • mm: Milímetros.
  • in: Pulgadas.
h1 { font-size: 24px; }
div { width: 10cm; }

Unidades Relativas

Las unidades relativas no están completamente definidas por sí mismas; su valor siempre se calcula referenciado respecto a otro valor. Son fundamentales para el Diseño Responsivo.

1. Relativas al Contenedor (%)

El porcentaje % se calcula en base al tamaño del elemento padre (contenedor directo). 

.hijo { width: 50%; } /* Ocupa la mitad de su padre */

2. Relativas a la Tipografía

Estas unidades se basan en el tamaño de la fuente:

  • em: Basada en la anchura/tamaño de la letra del elemento actual (o del padre si se usa en font-size).
  • rem: Basada en el tamaño de letra del elemento raíz (html). Es más predecible que em.
  • ex: Basada en la altura de la letra 'x' minúscula.
Diferencia EM vs REM:

Si el navegador tiene un tamaño base de 16px:

  • 1rem = 16px (Siempre).
  • 2em en un elemento con fuente de 10px = 20px.

Relativas al Viewport (Pantalla)

Estas unidades se basan en el tamaño de la ventana visible del navegador (Viewport).

Unidad Descripción
vw 1% de la anchura del viewport.
vh 1% de la altura del viewport.
vmin Toma el valor mínimo entre la altura y la anchura.
vmax Toma el valor máximo entre la altura y la anchura.
Ejemplo de uso (Pantalla completa): 
.hero-section {
    width: 100vw;
    height: 100vh; /* Ocupa toda la pantalla siempre */
}
