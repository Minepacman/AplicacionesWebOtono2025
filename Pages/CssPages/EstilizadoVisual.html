CSS Backgrounds

Las propiedades de fondo de CSS se usan para definir los efectos de fondo para los elementos.

Background Color

La propiedad background-color especifica el color de fondo de un elemento. Se pueden usar nombres de colores, valores Hexadecimales o RGB. 

h1.Tomato { background-color: Tomato; }
h1.RGBColor { background-color: rgb(30, 90, 120); }
h1.HEXColor { background-color: #2299DD; }

Background Image

La propiedad background-image especifica una imagen para usar como fondo de un elemento. Por defecto, la imagen se repite para que cubra todo el elemento. 

body {
    background-image: url("geometric-retro-pattern-background.png");
}

Repeat y Position

Background Repeat

Controla si la imagen se repite o no.

  • repeat-x: Se repite solo horizontalmente.
  • repeat-y: Se repite solo verticalmente.
  • no-repeat: La imagen se muestra solo una vez.
     body {
    background-image: url("geometric-retro-pattern-background.png");
       background-repeat: repeat;
       
}
ejemplo

Background Position y Attachment

La propiedad background-position especifica la posición inicial de la imagen (ej. right top, center).

La propiedad background-attachment: fixed hace que el fondo se quede fijo mientras el resto de la página hace scroll. 

body {
    background-image: url("imagen-centrada.webp");
    background-repeat: no-repeat;
    background-position: right top;
    background-attachment: fixed;
}

Bordes Redondeados (Border-Radius)

Con la propiedad border-radius puedes darle a cualquier elemento esquinas redondeadas.

Reglas de Valores

  • 4 valores: Arriba-Izq, Arriba-Der, Abajo-Der, Abajo-Izq.
  • 3 valores: Arriba-Izq, (Arriba-Der y Abajo-Izq), Abajo-Der.
  • 2 valores: (Arriba-Izq y Abajo-Der), (Arriba-Der y Abajo-Izq).
  • 1 valor: Todas las esquinas iguales.
#rcorners1 {
    border-radius: 15px 50px 30px 5px; /* 4 valores */
    background: #735021;
}

Fondos Múltiples

CSS permite agregar múltiples imágenes de fondo para un solo elemento, separándolas por comas. La primera imagen en la lista es la que está más "arriba" (más cercana al usuario). 

#example1 {
    background-image: url(img_flwr.gif), url(paper.gif);
    background-position: right bottom, left top;
    background-repeat: no-repeat, repeat;
}
