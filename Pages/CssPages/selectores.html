Sintaxis de CSS

Un conjunto de reglas CSS consiste en un selector y un bloque de declaración. 

h1 { 
    color: blue; 
    font-size: 12px; 
}

Selectores Básicos

1. Selector Universal (*)

* { margin: 0; }

2. Referencia por ID (#)

#articulo { font-size: 20px; }

3. Referencia por Clase (.)

.principal { font-size: 30px; }
/* Referencia compleja con etiqueta */
p.principal { font-size: 20px; }

Referencia por Atributo

Las especificaciones de CSS3 permiten referenciar por cualquier atributo.

/* Elemento p con atributo name="especial" */
p[name="especial"] {
    color: red;
}

Referencias Complejas

  • ^= : Comienza con el valor.
  • $= : Termina con el valor.
  • *= : Contiene el texto.
p[name^="mi"] { font-size: 20px; } /* Comienza con "mi" */
p[name$="mi"] { font-size: 40px; } /* Termina con "mi" */
p[name*="mi"] { font-size: 60px; } /* Contiene "mi" */

Pseudo-clases Estructurales

Nos permiten hacer referencias considerando la estructura del documento (hijos).

nth-child()

Permite encontrar un hijo específico o patrón (odd/even).

p:nth-child(2) { background: #999999; }

first-child, last-child, only-child

p:first-child { background: red; }   /* Primer hijo */
p:last-child  { background: blue; }  /* Último hijo */
p:only-child  { background: green; } /* Hijo único */

Selector Avanzado: Negación :not()

Se emplea para realizar una negación.

p:not(.classy) { color: red; }

Selectores Relacionales

1. Hijo Directo (>)

div > p.miestilo1 { color: #990000; }

2. Hermano Adyacente (+)

p.miestilo2 + p { color: #000099; }

3. Hermano General (~)

p.mitexto4 ~ p { color: #990099; }
