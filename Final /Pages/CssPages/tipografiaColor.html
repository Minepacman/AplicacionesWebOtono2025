Propiedades de Fuente

La propiedad font permite establecer de una sola vez los valores para todas las propiedades tipográficas en la hoja de estilo.

El orden de los valores es: font-style font-variant font-weight font-size font-family.

Nota: Las partes font-size y font-family son obligatorias. 

h1 {
    font: bold 20px verdana, sans-serif;
}  

h2 {
    font: bold 14px verdana, sans-serif;
}

Estilo y Variante

Font-style

Permite definir el aspecto de una familia tipográfica.

  • normal: Texto normal.
  • italic: Texto en cursiva.
  • oblique: Texto oblicuo (similar a cursiva).

Font-variant

Selecciona entre el aspecto normal y versalitas (mayúsculas pequeñas).

  • normal
  • small-caps: Transforma las minúsculas en mayúsculas de menor tamaño.

Grosor de Fuente (Font-weight)

Especifica el "peso" o grosor de la letra.

  • normal: Equivalente a 400.
  • bold: Negrita, equivalente a 700.
  • lighter / bolder: Más fino o más grueso que el elemento padre.
  • 100 - 900: Valores numéricos para mayor precisión.

Tamaño de Fuente (Font-size)

Permite establecer el tamaño del texto. Se pueden usar unidades de medida (px, em, %) o palabras clave.

Tamaños Relativos

Se usan palabras clave como larger y smaller. La referencia para calcular el tamaño es la letra del elemento contenedor (padre).

Familia Tipográfica (Font-family)

Establece el tipo de letra. Se puede indicar una lista de fuentes alternativas separadas por comas (Stack). El navegador intentará usar la primera; si no la tiene, probará la segunda, y así sucesivamente. 

body {
    font-family: Arial, Helvetica, Verdana, sans-serif;
}

p { 
    font-family: Georgia, Cambria, Times, serif;
}

Color de Texto

La propiedad color define el color del texto. Se pueden usar nombres predefinidos, valores Hexadecimales o RGB. 

h1 { color: Tomato; }
p.azul { color: DodgerBlue; }
p.verde { color: MediumSeaGreen; }

Color de Borde

También puedes establecer colores para los bordes utilizando la propiedad border o border-color. 

h2.Tomato { border: 2px solid Tomato; }
h2.DodgerBlue { border: 2px solid DodgerBlue; }
h2.Violet { border: 2px solid Violet; }
