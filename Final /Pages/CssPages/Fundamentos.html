Definición Básica de CSS

CSS es una tecnología complementaria al HTML5. Es un lenguaje que trabaja junto con HTML para proveer estilos visuales a los elementos del documento, como tamaño, color, fondo, bordes, etc.

CSS3 tiene la capacidad de sobrescribir los estilos básicos del navegador, crear nuevos estilos y combinarlos para hacer páginas atractivas.

Prefijos Importantes de Navegador:
  • -moz (Mozilla Firefox)
  • -webkit (Chrome, Safari)

Estructura de Bloque

Cuando hablamos de estructura, nos referimos a la manera en la que un sitio web está dispuesto. Cada navegador implementa una estructura por defecto.

Elementos Block

Son elementos posicionados uno sobre otro hacia abajo en la página. Este es el comportamiento por defecto en los navegadores.

Elementos Inline-block

Son elementos posicionados lado a lado, uno al lado del otro en la misma línea, sin ningún salto de línea a menos que ya no haya más espacio horizontal para ubicarlos.

Definición de Elementos Estructurales

Esto es con la finalidad de que el navegador maneje todas las estructuras de igual manera (Reset básico). 

* {
    margin: 0px;
    padding: 0px;
}
header, section, article, footer, nav, aside, figure, figcaption, hgroup {
    display: block;
}

Display: None

El valor más sencillo de display es none. Hace que el elemento no genere ninguna caja; desaparece por completo de la página y no ocupa sitio.

Modelo de Caja

Se le denomina Modelo de Caja a todas las reglas que especifican la disposición en la que será organizada una página web.

Métodos de Inserción de Estilos

1. Estilos en Línea (Style)

Es una forma simple y rudimentaria, considerada poco práctica. 

<p style="font-size: 40px"> Esto es un párrafo </p>

2. Estilos Embebidos (Head)

Se coloca el estilo entre las etiquetas <style> dentro del <head>. 

<head>
    <style>
        p { font-size : 40px }
    </style>
</head>

3. Archivos Externos (Link)

Facilita la actualización de los estilos. Se implementa mediante el uso de la etiqueta <link>. 

<head>
    <link rel="stylesheet" type="text/css" href="css/estilos.css">
</head>
